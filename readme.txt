Running front end:
1. Load index.html into web browser.
NOTE: Do not do this from file directory, it will throw a file ownership error.

Running back end: 
1. Open terminal and type out "node app.js"

Working with the web service:
1. Ensure both frontend and backend are running.
2. Open your web browser and go to the frontend URL (e.g., http://localhost:8000/index.html).
3. Enter the required input values (Age, Height, and Weight) into the respective fields.
4. Click the "Calculate" button.
5. View the results in the modal dialog.
